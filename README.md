# Uzum · дашборд карточек (UytMarket)

Интерактивный дашборд состояния товарных карточек на Uzum Market: динамика продаж,
ранг, остатки, воронка и готовые сигналы к действию по каждой карточке.

**Открыть:** [index.html](index.html) — самодостаточная страница, скачать и открыть
двойным кликом. Внешних запросов не делает, данные вшиты внутрь.

> ⚠️ Репозиторий **приватный**: страница содержит выручку, прибыль, остатки и цены
> по каждому товару. Не делать публичным без осознанного решения — данные попадут
> в кэш поисковиков и останутся там после удаления.

## Что внутри

| Файл | Что это |
|---|---|
| `index.html` | дашборд на дату последнего прогона (данные вшиты) |
| `data/portfolio_log.csv` | append-only лог снимков каталога: строка на (дата × карточка) |
| `data/sales_daily.csv` | продажи по дням: штуки, выручка, прибыль, заказы, отмены |
| `data/keys_manual.csv` | позиции по поисковым ключам — заполняется вручную |

История коммитов = архив ежедневных снимков: любой день восстанавливается из git.

## Как обновляется

Локальный launchd-агент `com.uzum.monitor.portfolio` ежедневно в **07:10 KST**:

1. `uzum_portfolio.py` — тянет Uzum Seller API (каталог + финансы), пишет CSV и xlsx;
2. `build_dashboard_html.py` — пересобирает `index.html` из CSV;
3. `deploy_dashboard.sh` — коммитит и пушит сюда (пустой diff → коммита нет).

Источник скриптов: локальный скилл `uzum-card-monitor`. Токен продавца живёт только
в `~/.config/uzum/`, в репозиторий не попадает.

## Границы данных

- Метрики каталога (ранг, просмотры, конверсия, % возвратов) — **скользящие**
  показатели API, поэтому фиксируются с датой снимка.
- Выручка и прибыль считаются **без отменённых** заказов;
  прибыль = выручка − комиссия − логистический сбор (из `/v1/finance/orders`).
- Позиции по поисковым ключам **не автоматизируются** (витрина Uzum закрыта капчей,
  у ZoomSelling нет API) — это ручной еженедельный срез в `data/keys_manual.csv`.
- Магазин Techno_shop (44143) с 21.07.2026 отдаёт «Shop is not available»,
  поэтому в дашборде только UytMarket (112728).
