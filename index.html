Правки и эффект

Ежедневное обновление 07:10 KST из Uzum Seller API (ранг, продажи, остатки, цены, просмотры, конверсия). Просмотры в API — накопительный счётчик, поэтому трафик за период считается как прирост между снимками лога; клики API отдаёт пустыми, поэтому CTR не показывается. «Приз» — размер разрыва до медианы портфеля при текущем трафике, а не прогноз эффекта правки: сколько даёт конкретная правка, накапливается в блоке «Правки и эффект». Позиции по поисковым ключам и данные по конкурентам не автоматизируются (витрина за капчей, у ZoomSelling нет API) — еженедельный ручной срез. Отменённые заказы в продажи не входят.
